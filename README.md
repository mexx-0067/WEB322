# WEB322 - Web Programming Using JavaScript

## Overview
This repository contains materials, practice exercises, labs, and assignments for the **WEB322** course. It focuses on server-side JavaScript development using Node.js, Express, and related technologies.

## Repository Structure
- **WEB322-Course/**: Main course materials
- **WEB322-practice/**: Practice exercises and examples
- **week*/**: Weekly labs and lecture notes
- **assignment*/**: Assignment submissions
- `index.js`, `objects.js`, etc.: Core scripts and examples

## Technologies
- **JavaScript** (Node.js)
- **Express** / EJS templating
- HTML, CSS

## Getting Started
```bash
# Clone the repo
git clone https://github.com/mexx-0067/WEB322.git

# Navigate to a practice folder (example)
cd WEB322/WEB322-practice/WEB322-practice

# Install dependencies (if applicable)
npm install

# Run examples
node index.js
